# Wyatt Yousey Portfolio

A modern developer portfolio built to showcase my transition into software engineering, technical projects, education, and growth as an engineer.

This project serves as both a personal portfolio and a demonstration of my frontend development skills. The application was designed from the ground up with a strong emphasis on clean UI, reusable architecture, responsive design, and engaging user interactions.

---

## About

After spending several years in the manufacturing industry working in welding and quality control, I transitioned into software engineering in search of a more challenging and rewarding career path. Since beginning that journey, I have completed a comprehensive software engineering program through TripleTen, numerous independent courses, and built a growing collection of projects focused on web development, automation, architecture, and problem solving.

This portfolio was created to tell that story while demonstrating the technical skills I've developed along the way.

## Architecture

One of the primary goals of this project was maintaining a scalable structure that would be easy to expand as my portfolio grows.

### Project Structure

```text
src
│
├── assets
├── components
│   ├── cards
│   ├── layout
│   └── ui
│
├── data
│
├── pages
│
├── routes
│
└── styles
```

### Data-Driven Design

Most content is stored inside dedicated data files rather than directly inside components. This allows:

- Easier maintenance
- Reusable UI components
- Cleaner page logic
- Faster future updates

Examples include:

- Experience Timeline
- Skills Data
- Project Data
- Statistics Data
- Contact Information

---

## Design Goals

The visual design centers around a dark theme accented with bright highlight colors that guide user attention and create visual identity throughout the application.

Each page was designed with its own animation personality:

| Page | Animation Style |
 | -------- | ---------------- |
 | Home | Dynamic asset assembly |
 | Experience | Smooth overshoot and settle |
 | Skills | Staggered entrance animations |
 | Projects | Spotlight transitions |
 | Contact | Alternating conversational motion |

The goal was to create a portfolio that feels interactive and memorable without becoming distracting.

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
